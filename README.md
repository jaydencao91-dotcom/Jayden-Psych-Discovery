### 🌊 溯源与初衷 / The Origin
我们每天都在做成千上万个决策，却很少意识到大脑在幕后如何运行。这个仓库是我个人的“认知实验室”——在这里，我通过英文学术前沿与心理学实证，去拆解那些看起来“理所当然”的日常行为。

We make thousands of decisions daily, yet rarely notice how our brains operate behind the scenes. This repository is my personal "Cognitive Lab"—a curated collection of psychological insights to deconstruct the "obvious" in our lives.
